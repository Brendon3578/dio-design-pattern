# Projeto Spring Boot Design Pattern 🍃

Projeto feito duranto o Bootcamp GFT Start #3 Java da [Digital Inovation One](https://www.dio.me/)

🔮 Feito para estudar **Design Patterns** (Singleton, Strategy, Facade) e **Spring Framework**

## 🎈 Descrição
A ideia do projeto é ser um API REST com as integrações do *Spring Data JPA* e a API externa *ViaCEP*

## 🧱 Ferramentas utilizadas

- Java 11
- IntelliJ
- **Spring Web**
- **Spring Data JPA**
- **Open Feign**
- **H2 Database**
